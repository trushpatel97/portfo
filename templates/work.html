001 : Color Proportions

Summary

This program goes through every pixel of an image or video and outputs the color proportions. The results are displayed in three formats: bar graph, 3D cluster, and in a single line (least to most dominant color). Of course with a video, each frame might have a different color proportion. Therefore, we made a program that updates the graphs according to each new frame.

One of the hardest parts about this project was the phase of taking a video, and putting every frame's result and adding it in a single line. To overcome this obstacle we contacted an App Developer who inspired our project. He was able to give us a snippet of his code to use. Although the code he gave to us was in a different language, we were able to read through the code and build upon it.
